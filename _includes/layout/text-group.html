Text Group

Text groups are an implantation of base layout classes; no new styles are introduced here. Often there is a need to control how text breaks across lines. To solve this, groups of text must be wrapped in span elements and layout classes applied. This need is common enough to warrant it's own set of examples for demonstration and testing purposes.
