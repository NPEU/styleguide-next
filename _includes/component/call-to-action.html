Call to Action (CTA)

The CTA is a prominent link. It's NOT a button. It looks a bit like a button but it's a link.

CTA - standard

Call to Action

Show code: 
<a href="#" class="c-cta">
    Call to Action
</a>

CTA - with icon

Call to Action       Search

Note that some icons fill their 'square' better than others, so it's not possible to use CSS to 'nicely' space icons from the text, so in the Search example above a &nbsp; is placed between the icon and the text.

Show code: 
<a href="#" class="c-cta">
    Call to Action<svg focusable="false" aria-hidden="true" width="1.25em" height="1.25em"><use xlink:href="#icon-chevron-right"></use></svg>
</a>
...
<a href="#" class="c-cta">
    <svg focusable="false" aria-hidden="true" width="1.25em" height="1.25em"><use xlink:href="#icon-search"></use></svg>&nbsp;Search
</a>
