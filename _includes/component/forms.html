Tool Form IN PROGRESS

Simple forms for things like filtering lists. Note it's quite likely a wrapping fieldset isn't appropriate.

This example includes Filterability attributes that might be relevant here, but not EVERYTHING Filterability would need to function.

Note the fieldset's div is so flex works properly - it doesn't in Chrome and older versions of Firefox.
