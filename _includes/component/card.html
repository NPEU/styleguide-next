Card

Cards are more complex than you might think.

  1. Firstly, there are several components to them, and only the title and link are required. The others can appear or not in any combination.
  2. Next, they need to be able to account for these variations inside a layout of some kind and present themselves smartly in relation to each other.
  3. Then the heading levels need to make sense in their surrounding context.
  4. Finally, interaction with them (the links and state styling) should make sense in accessibility terms, as we all in NoCSS scenarios.
